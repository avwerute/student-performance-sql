# Student Performance Analysis (MySQL)

## Tools
- MySQL
- MySQL Workbench

## Project Overview
This project analyzes student academic data to identify performance trends and high-achieving students.

## Key Tasks
- Filtered student records by GPA and school lunch eligibility
- Calculated average GPA by grade level
- Identified grade levels below a GPA benchmark
- Joined student and grade tables using a LEFT JOIN

## SQL Concepts Used
- SELECT, WHERE, ORDER BY
- DISTINCT
- GROUP BY, HAVING
- LEFT JOIN
